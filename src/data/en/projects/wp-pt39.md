## What is WP PT39?

WP PT39 is a professional WordPress theme built for editorial outlets, digital magazines, and news sites. A fully Customizer-configurable homepage, native light/dark mode, advanced cookie consent management, and built-in SEO features — all written in plain PHP with no extra frameworks, for maximum lightness and WordPress compatibility.

## Key Features

- **Modular homepage** — featured-articles hero, real-time news ticker, up to 15 configurable category sections, an "Inspiration" carousel, and a YouTube playlist block
- **Light/dark mode** — persisted preference, dual logo and header images, dynamic browser theme-color updates
- **GDPR cookie consent** — banner and preferences modal with 4 categories, deferred scripts and iframes, custom event for integrations
- **Custom XML sitemap** — index, posts, pages, categories, and tags, cached via transients with automatic invalidation
- **REST API extensions** — reading time, view count, and category details added to the posts endpoint
- **Editorial metrics** — anti-inflation view counter and estimated reading time
- **Accessible UX** — gesture-driven mobile menu, keyboard-navigable submenus, image lightbox, scroll indicator, and AJAX login modal

## Technology

Built with plain PHP, custom CSS, and vanilla JavaScript — no external frameworks. Requires WordPress 5.9+ and PHP 7.3+.

## Website

The theme powers [pt39.it](https://www.pt39.it), the news portal of Circolo Socio-Culturale Palazzo Tenta 39.
